RenoirSt

A DSL enabling programmatic cascading style sheet generation for Pharo Smalltalk

View project onGitHub

Intro

This MIT licensed library is aimed to provide a DSL to easily support CSS generation for Pharo.

Checkout the installation instructions in the GitHub project or the online-tutorial

Goals

  • Improve CSS integration with existing Web Frameworks
  • Write & refactor in Smalltalk, deploy to CSS

Features

  • Common properties declaration
  • CSS3 Selectors Support
  • Important Rules
  • Media Queries Support

Authors
