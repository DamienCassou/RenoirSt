Renoir.St
=========

*A DSL enabling programmatic cascading style sheet generation for Pharo Smalltalk*

##Goals
- Improve CSS integration with existing Web Frameworks
- Write in Smalltalk deploy to CSS

###Highlights:
- **License**: MIT
- **Supported Platforms**: Pharo 3
